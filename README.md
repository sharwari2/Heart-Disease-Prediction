# Heart Disease Prediction

A machine learning web application that predicts the likelihood of heart disease based on patient data.

## Tech Stack
- Python
- Streamlit (Frontend)
- Scikit-learn / PyTorch (ML Model)

## Features
- Predicts heart disease risk from user inputs
- Trained on medical dataset
- Interactive web interface

## How to Run
1. Clone the repository
2. Install dependencies:
```
   pip install -r requirements.txt
```
3. Run the app:
```
   streamlit run app.py
```

## Project Structure
- `frontend/` - UI code
- `backend/` - Model inference logic
- `data/` - Dataset
- `saved_models/` - Trained model files
